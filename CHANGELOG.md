## [1.0.0] — 2026-04-06

### Added
- `/playcraft:store-listing` — generate complete Play Store title, short description, and full description with character limit enforcement
- `/playcraft:release-notes` — write user-friendly release notes under 500 characters
- `/playcraft:aso-audit` — score listing out of 30 across title, short desc, and full desc with keyword recommendations
- `/playcraft:declarations` — generate complete Play Console policy declarations checklist (data safety, financial, AI content, families, identity)
- `/playcraft:screenshots` — write captions (~30 chars), alt text, and slot recommendations for up to 8 screenshots
- `/playcraft:setup` — show current plugin config and how to update it
- `aso-agent` — keyword research and density analysis for Google Play ASO
- `listing-writer` — specialist copywriter for Play Store titles and descriptions
- `policy-checker` — pre-submission policy compliance review against Play Store policies
- `locale-agent` — localization for 14 locales including Southeast Asia priority (th-TH, id-ID, ms-MY, zh-TW, zh-CN, ja-JP, ko-KR) and global tier 2
- `play-policy-2026` skill — 2025–2026 Play Store rules, character limits, and new requirements
- `closed-testing` skill — closed testing requirements for new developer accounts (12 testers, 14 days)
- `financial-declarations` skill — financial features declaration walkthrough with Play Console navigation paths
- `ai-content-declarations` skill — AI-generated content declaration guide (2025 requirement)
- `developer-identity-th` skill — identity verification and tax setup for Thailand-based developers
- `pre-release-check` hook — pre-flight checklist reminder before release builds
- `userConfig` integration — developer details set once at install, used globally across all projects
